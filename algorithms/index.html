Algoritmi za sortiranje nizova

Maturski rad, Lav Leon Hudak, 2018/2019

Nizovi su struktura podataka koja se sastoji iz više promenljivih, od kojih svaka ima svoj index u nizu. Najprostiji nizovi su jednodimenzionalni, dok, kada bismo imali beskonačno mnogo memorije i kompjuterske snage, mogli bi smo da imamo n-todimenzionalni niz. Kada pravimo niz, obično je potrebno definisati vrstu promenljivih koje će biti u nizu. Postoje različite vrste nizova, kao što su:

  • Vector
  • List
  • Stack
  • Queue
  • Matrix
  • Dope Vector
  • Dynamic Array
  • Parallel Array
  // sintaksa nizova u različitim programskim jezicima
  int foo [5] = { 16, 2, 77, 40, 21 }; // C++
  int[] foo = new int[5]; // C#, nepopunjeni niz dužine 5
  let foo = [1, 2, 3, 4, 5]; // JS, nije potrebno deklarisati vrstu promenljive              
  foo = ["jabuka", "banana", "lubenica"] // Python

Višedimenzionalni nizovi su zapravo nizovi nizova. Svaki element u nizu je niz, koji sadrzi neke vrednosti. 2D niz, takođe pod imenom "matrica" (engl. "Matrix") je bitan koncept u matematici, a samim time i u teoriji kompjutera i svetu programiranja.

Bubble Sort

Prosečna složenost: O(n2)

Najprostiji algoritam za sortiranje niza. Prolazi kroz niz i kada naiđe na par elemenata koji nisu poređani, zameni im mesta.

Insertion Sort

Prosečna složenost: O(n2)

Prost algoritam za sortiranje koji postepeno pravi sve veću i veću sortiranu listu od niza. Neefikasan za veće nizove.

Selection Sort

Prosečna složenost: O(n2)

Polako povećava sortirani podniz tako što iz nesortiranog podniza uzima najmanji element i stavlja ga na kraj sortiranog podniza.
